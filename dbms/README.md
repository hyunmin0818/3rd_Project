# DBMS 릴레이션 매핑

## User테이블: 사용자 정보를 저장하기 위한 테이블

- **사용자ID(PK)**
- **이름**
- **연락처**
- **비밀번호**
- **이메일**
- **생년월일**
- **성별**
- **찜(관심)목록**

## Operator테이블: 운영자 정보를 관리하기 위한 테이블

- **운영자ID(PK)**
- **이름**
- **직책**
- **연락처**
- **생년월일**

## Movie테이블: 영화의 정보를 관리하기 위한 테이블<br>(굵은글씨API제공정보)

- 영화포스터
- 영화소개
- **영화이름**
- **영화이름(영문)**
- **영화코드**
- **제작연도**
- **제작국가**
- **유형**
- **장르**
- **제작상태**
- **감독**
- **제작사**
